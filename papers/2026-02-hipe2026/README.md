### 关于作者

**Juri Opitz**（第一作者）是苏黎世大学计算语言学系的研究员，研究方向包括自然语言处理、语义评估和信息抽取。**Simon Clematide**（通讯作者）是苏黎世大学计算语言学系的高级研究员，长期参与 CLEF 系列评测活动的组织工作。该团队横跨瑞士三所顶级研究机构，在多语言 NLP 和数字人文交叉领域有深厚积累。

### 关于来源

**Anthropic** 是 Claude 系列大语言模型的开发公司，由前 OpenAI 研究副总裁 Dario Amodei 和 Daniela Amodei 于 2021 年创立。Claude Code 是 Anthropic 推出的命令行 AI 编程助手，支持钩子（Hooks）、扩展（Extensions）、子代理（Sub-agents）等高级特性，是目前功能最完整的 AI 编码工具之一。

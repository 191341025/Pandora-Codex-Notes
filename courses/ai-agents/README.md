### 关于来源

**Google Cloud** 是全球三大云计算平台之一，其 AI/ML 产品线包括 Vertex AI、Gemini API、Agent Development Kit (ADK) 等。这份面向创业公司的技术指南系统梳理了 AI Agent 的完整技术栈——从基础组件到生产部署，是入门 Agent 生态的优质参考。

### 关于来源

**Martin Kleppmann** 是剑桥大学分布式系统研究员，DDIA 被公认为数据系统领域的"圣经级"著作。全书从单机存储引擎讲到分布式共识算法，覆盖了数据密集型应用的完整技术栈。虽然出版于 2017 年，但其核心原理（复制、分区、事务、一致性模型）至今仍是理解现代数据系统的基石。

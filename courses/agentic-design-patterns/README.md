### 关于作者

**Antonio Gulli** 是 Google 工程副总裁，长期深耕人工智能与机器学习领域。他在搜索引擎、大规模分布式系统和深度学习方面有超过 20 年的工程经验，著有多本技术书籍。本书系统梳理了 Agent 系统中反复出现的设计模式，是目前最全面的 Agentic 设计模式参考之一。

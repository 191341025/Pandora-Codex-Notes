### 关于来源

**Google Agent Development Kit (ADK)** 是 Google 推出的开源 Agent 开发框架，支持多 Agent 编排、会话状态管理、工具集成和流式输出等特性。本教程是官方动手实践指南，从 Hello World 级别的天气查询 Agent 逐步深入到多 Agent 协作、回调机制和高级工具使用。

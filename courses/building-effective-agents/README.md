### 关于作者

**Erik Schluntz** 和 **Barry Zhang** 是 Anthropic 的工程师，专注于 Agent 系统的设计与实现。这篇白皮书是 Anthropic 官方对"如何构建有效 Agent"的系统性总结，从增强型 LLM 到 Workflow 模式再到自主 Agent，给出了极具实操价值的架构建议，发布后在开发者社区引发广泛讨论。

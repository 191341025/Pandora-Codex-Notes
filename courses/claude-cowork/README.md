### 关于来源

**Anthropic** 是 Claude 系列大语言模型的开发公司，由前 OpenAI 研究副总裁 Dario Amodei 和 Daniela Amodei 于 2021 年创立。Cowork 是 Anthropic 推出的研究预览功能，让 Claude 能够在沙盒环境中自主使用浏览器和计算机完成知识工作任务，如数据分析、市场调研和文档整理。

### 关于来源

**Mintplex Labs** 是 AnythingLLM 的开发团队。AnythingLLM 是一个开源的全栈 LLM 应用平台，支持多种模型接入（OpenAI、Anthropic、本地模型等）、文档 RAG、AI Agent、工作流编排等功能，在 GitHub 上获得了超过 30k Star，是目前最受欢迎的开源 LLM 应用框架之一。

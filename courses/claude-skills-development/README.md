### 关于来源

**Anthropic** 是 Claude 系列大语言模型的开发公司，由前 OpenAI 研究副总裁 Dario Amodei 和 Daniela Amodei 于 2021 年创立。Skills 是 Claude 生态中的可复用能力模块，允许开发者将特定领域的专业知识和工作流封装为可分享的技能包。
